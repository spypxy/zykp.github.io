山楂麦芽茶

分类：老年养生药膳 | 阅读量：91123

山楂麦芽茶是一道传统的消食化积、健脾开胃养生茶饮。山楂性微温味酸甘，具有消食健胃、行气散瘀的功效；麦芽性平味甘，具有消食健胃、回乳消胀的功效。两者合用，特别适合食积不化、脘腹胀满、消化不良的人群饮用。这道茶饮酸甘可口，老少皆宜，是日常消食健胃的佳品。

原料准备

1. 山楂：30克

2. 麦芽：30克

3. 红糖：适量（约20克，可根据口味调整）

4. 清水：1000毫升

制作方法

1. 处理山楂：将山楂洗净，切成两半，去除果核。如果使用的是干山楂片，直接使用即可。

2. 处理麦芽：将麦芽洗净，沥干水分备用。

3. 煮茶：将山楂和麦芽放入锅中，加入清水，大火煮沸后转小火继续煮15-20分钟。

4. 调味：关火后加入红糖搅拌至融化，焖5分钟即可饮用。

5. 代茶饮用：可将茶汤倒入保温杯中，代茶饮用，一日内饮完。

功效与作用

1. 消食健胃：山楂具有消食健胃的功效，尤其善于消肉食积滞；麦芽善于消米面薯芋类积滞。两者合用，可以消除各种食物积滞。

2. 行气散瘀：山楂具有行气散瘀的功效，能够活血化瘀，对于瘀血所致的痛经、产后腹痛等有一定缓解作用。

3. 回乳消胀：麦芽具有回乳消胀的功效，哺乳期妇女断奶时适量饮用可以帮助回乳。

4. 健脾开胃：这道茶饮能够刺激胃液分泌，增进食欲，对于食欲不振、消化不良有很好的改善作用。

适用人群

1. 食积不化者：暴饮暴食后出现脘腹胀满、嗳气反酸者

2. 消化不良者：长期消化不良、食欲不振者

3. 高血脂患者：山楂具有降血脂的作用，适量饮用对高血脂有一定辅助治疗作用

4. 产后妇女：产后需要回乳的妇女

5. 产后瘀血：产后恶露不尽、瘀血腹痛者（可去红糖饮用）

注意事项

1. 胃酸过多者慎用：山楂味酸，胃酸过多、胃溃疡患者不宜多饮。

2. 孕妇慎用：山楂有活血作用，孕妇不宜大量饮用，以免刺激子宫。

3. 哺乳期妇女：如果需要继续哺乳，不宜饮用此茶，以免回乳。

4. 糖尿病患者：应减少红糖用量或不放红糖。

5. 儿童：儿童胃肠道功能较弱，不宜过量饮用。

6. 空腹饮用：不宜空腹饮用，以免刺激胃黏膜。

山楂麦芽茶是一款适合日常饮用的养生茶饮，特别适合节假日暴饮暴食后消食解腻。但应注意根据自己的体质情况适量饮用，不可过量。如有严重的消化系统疾病，应及时就医治疗。
